# 🤖 ApexPlanet Data Analytics Internship – Task 4

## 📌 Project Title

**Advanced Analytics & Sales Prediction using Machine Learning**

---

## 📖 Overview

This project is completed as part of the **ApexPlanet Software Pvt. Ltd. Data Analytics Internship – Task 4**.

The main objective of this task is to apply machine learning techniques to analyze supermarket sales data and build a predictive model. The project focuses on data preprocessing, feature engineering, model training, evaluation, and saving the trained model for future predictions.

---

## 🎯 Objectives

* Understand and prepare data for machine learning.
* Convert categorical data into numerical format.
* Split data into training and testing sets.
* Build a predictive machine learning model.
* Evaluate model performance.
* Save the trained model for reuse.

---

## 📂 Dataset

**Dataset Name:** SuperMarket Analysis Dataset

The dataset contains supermarket transaction information:

* Invoice ID
* Branch
* City
* Customer Type
* Gender
* Product Line
* Unit Price
* Quantity
* Payment Method
* Sales
* Rating

## 🤖 Machine Learning Model

### Algorithm Used

**Linear Regression**

The model was trained to predict supermarket sales based on customer and product-related features.

---

## 📊 Model Evaluation Metrics

The model performance was evaluated using:

* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* R² Score

---

## 📈 Visualization

Created visualization for:

* Actual Sales vs Predicted Sales

This helped analyze the performance of the trained model.

---

## 💾 Model Saving

The trained model was saved using Pickle:

`sales_prediction_model.pkl`

This allows the model to be reused without retraining.
